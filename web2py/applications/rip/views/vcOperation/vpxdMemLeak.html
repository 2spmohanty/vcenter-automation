VPXD Memory Leak

URI: http://{{=response.app_uri}}/rip/vc_0/vpxdmemleak
This API can be used to identify memory leaks in VPXD service. The crux of this API is CHAP and AH64, developed by TIM BODDY, VMWare. More details on CHAP would be available in documentation page, where Tim discusses the advantages of using it for Memory analysis. A livecore dump is taken on the service and anlaysis is done using different debuggers to gather the output. This API runs with a disclaimer and caution as coredump might affect the memory/process for quite some time and it can disrupt some critical programs running on the VC. So run it when it is absolutely required to go for memory analysis. Please refer sample JSON.
