VM Power Operation

URI: http://{{=response.app_uri}}/rip/vm_0/vmpowerops
This API can be used to trigger Power Operation on VMs. The "ops" key can be set to on | off | reset. Additionally, instead of "vm" key the "pattern" key can be used, where in the Power operation gets triggered on VMs whose name contains the pattern string. The "pattern" and "vm" key is mutually exclusive. Either one needs to be present. The "pattern" key overrides the "vm" key
